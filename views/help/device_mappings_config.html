Device Mappings Configuration Format

The device mappings configuration file should be a CSV file in the format below.

Device Context Camera
Device1 ID a1b2c3d4-5e6f-47g8-9h0i-j1k2l3m4n5o6
Device2 ID b2c3d4e5-6f7g-89h0-i1j2-k3l4m5n6o7p8

  • Device: The identifier of this unique device passed as a the value of a dedicated JSON key
  • Context Camera: The unique ID of the context camera from your command org. The Camera ID can be found in the browser URL when the camera is opened in Command or using the Get Camera Data API endpoint
